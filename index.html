Assunto que mais gostamos

CPU- Gostei do conteúdo pois a CPU é certa forma o cérebro do computador, é por lá que é executado os programas, fazendo os três passos (buscar, ecodificar, executar) em um formato cíclico. Vale destacar as duas arquiteturas dos processadores a arquitetura CISC e RISC, a primeira com o objetivo de que o processador possa realizar todos os tipos de tarefas, já o segundo é voltado para tarefas especificas.

-Luiz Fabiano Mineiro

CPU-Canaltech

Linguagens de Programação-Gostei do conteúdo de linguagens de programação pelo simples fato de ser um meio de transmitir informação. Por se tratar de linguagens, servem para comunicação de humanos com máquinas. É bastante interessante entender sobre cada tipo de linguagem da programação e suas funcionalidades. O assunto me chama bastante atenção,assim buscarei entender um pouco sobre as principais linguagens (Python, Java, C++ (e C), JavaScript e Go), atualmente tive contato com poucas linguagens, sendo elas a linguagem C, HTML e CSS.

-Maria Beatriz Rodrigues
